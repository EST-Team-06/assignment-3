# Assignment 3
## Markdown to PDF
To use the script `md_to_pdf.py`, make sure you have the following installed:
```
pip install markdown weasyprint pygments
```
Then run it via:
```
python md_to_pdf.py  tasks_formal_modelling/answers.md -o tasks_formal_modelling/answer.pdf
```
## Use of AI
* The `md_to_pdf.py` script was vibe coded since it has zero relevance to the assignment, it's a tool with the sole purpose to get our Markdown into the submission format PDF.
